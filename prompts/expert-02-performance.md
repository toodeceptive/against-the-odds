# Expert 2: Performance & Optimization Guru

**Domain**: Performance optimization, resource efficiency, scalability, bottlenecks  
**Skills**: Profiling, benchmarking, algorithm optimization, caching strategies, bundle optimization  
**Objective**: Identify and optimize all performance bottlenecks across code, builds, tests, and workflows

## Focus Areas

### 1. Code Performance

- [ ] Profile JavaScript/TypeScript execution (CPU, memory)
- [ ] Identify slow algorithms and optimize
- [ ] Review async/await patterns for efficiency
- [ ] Assess memory leaks and optimize
- [ ] Review loop performance and optimize
- [ ] Assess function call overhead

### 2. Bundle Optimization

- [ ] Analyze bundle sizes (webpack/vite output)
- [ ] Implement tree-shaking where missing
- [ ] Review code splitting strategies
- [ ] Optimize imports (avoid full library imports)
- [ ] Assess and optimize asset loading
- [ ] Review and optimize image compression

### 3. Database & Query Optimization

- [ ] Profile database queries (if applicable)
- [ ] Review and optimize indexes
- [ ] Identify N+1 query problems
- [ ] Optimize join operations
- [ ] Review query caching strategies
- [ ] Assess connection pooling

### 4. API Performance

- [ ] Review API call patterns (batching opportunities)
- [ ] Assess request/response sizes
- [ ] Review caching strategies (HTTP, application-level)
- [ ] Optimize serialization/deserialization
- [ ] Review rate limiting impact
- [ ] Assess API response times

### 5. Build & Test Performance

- [ ] Profile build times
- [ ] Optimize build configuration (caching, parallelization)
- [ ] Review test execution times
- [ ] Optimize test setup/teardown
- [ ] Assess CI/CD job performance
- [ ] Review dependency installation times

### 6. Frontend Performance

- [ ] Assess page load times
- [ ] Review rendering performance
- [ ] Optimize CSS (critical CSS, unused CSS)
- [ ] Review JavaScript execution blocking
- [ ] Assess lazy loading implementation
- [ ] Review resource prioritization

### 7. Workflow Performance

- [ ] Profile GitHub Actions workflow execution
- [ ] Optimize workflow dependencies
- [ ] Review caching strategies in workflows
- [ ] Assess parallel job opportunities
- [ ] Optimize artifact handling
- [ ] Review workflow timeouts

## Deliverables

1. **Performance Audit Report** (`docs/expert-analyses/expert_02_performance_audit_YYYYMMDD.md`)
   - Performance bottlenecks identified
   - Benchmark results (before/after)
   - Optimization recommendations
   - Performance metrics and targets

2. **Optimization Recommendations**
   - Critical optimizations (immediate impact)
   - High-value optimizations (significant improvement)
   - Nice-to-have optimizations (marginal gains)

3. **Performance Monitoring Setup**
   - Metrics collection
   - Performance dashboards
   - Alerting thresholds

4. **Performance Benchmarks**
   - Baseline measurements
   - Target performance goals
   - Optimization tracking
