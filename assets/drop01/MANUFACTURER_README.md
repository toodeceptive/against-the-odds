# AO Drop 01 — For Manufacturers

This folder contains spec packs and artwork for **Against The Odds** Drop 01. We are ordering **1 sample each** of 4 products (beanie, fitted flatbrim cap, hoodie, long sleeve) from **2 manufacturers** for comparison.

## What You’ll Find Here

- **spec-pack/** — Per-product spec templates (beanie, cap, hoodie, long sleeve) and a master README. Fill in your details and export to PDF for submission.
- **artwork/** — Place vector (`.ai` / `.pdf` / `.svg`) and 300 DPI PNG by product and placement (front, back, sleeve, etc.). See `docs/launch/07_drop01_product_image_plan_extended.md` for exact layout.
- **exports/** — Print-ready PNG previews at final print size (300 DPI).
- **mockups/** — Visual mockups per product for approval.

## Full Plan Reference

- **Extended plan (all 4 products)**: `docs/launch/07_drop01_product_image_plan_extended.md` (in repo root).
- **Hoodie-only plan**: `docs/launch/07_drop01_product_image_plan.md`.
