**Date**: 2026-01-29

## Overview

This directory contains print specification templates for all 4 products in Drop 01:

1. **Beanie** (`beanie_spec_template.md`)
2. **Fitted Flatbrim Cap** (`cap_spec_template.md`)
3. **Hoodie** (`hoodie_spec_template.md`)
4. **Long Sleeve Shirt** (`longsleeve_spec_template.md`)

---

## Usage

### For Manufacturers

1. **Review Templates**: Each template contains complete specifications for one product
2. **Fill in Details**: Replace placeholder values (Pantone colors, exact measurements, etc.) with final values
3. **Convert to PDF**: Convert markdown templates to PDF for professional presentation
4. **Include Artwork**: Reference artwork files from `../artwork/` directory
5. **Include Mockups**: Reference mockup images from `../mockups/` directory

### For Production Team

1. **Customize Templates**: Update templates with final artwork dimensions and colors
2. **Generate PDFs**: Convert to PDF format for manufacturer distribution
3. **Package Spec Pack**: Combine PDFs, artwork files, and mockups into complete spec pack
4. **Send to Manufacturers**: Distribute complete spec pack to both manufacturers

---

## Template Structure

Each template includes:

- **Product Information**: Garment type, size, color, material
- **Design Placements**: Detailed specifications for each placement (front, back, sleeve, etc.)
- **Artwork Files**: References to vector and raster artwork files
- **Print Method Notes**: Recommended print methods and special instructions
- **Color Specifications**: Pantone/HEX color definitions
- **Quality Requirements**: Durability, visibility, placement accuracy requirements
- **Mockup References**: Links to mockup images
- **Manufacturer Confirmation Checklist**: Items for manufacturer to confirm
- **Notes**: Additional production notes and considerations
